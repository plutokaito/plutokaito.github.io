Mac 上常用的软件

Tue, Jun 8, 2021

该篇文章主要是主要记录自己在 mac 上常用的软件

1. 聊天工具

  • wechat(微信)
  • 钉钉
  • Skype

2. 开发工具

idea社区版 Java 开发工具，能满足日常开发。

vscode 一款不错的编辑器，我目前用来开发 PHP 和一些常用的脚本编辑

iterm2 命令行终端

oh my zsh 命令行 shell

JDK Mission Control Java 8 这个需要账号登录并下载。第一次可以从官网下载，下载完成后放入自己的网盘或者 U 盘中，这样不用每次都通过官网的方式查找。

docker

  • docker ui docker

DBeaver 数据库客户端，用 Java 编写，由于自身需要用 Java 进行开发，所以使用了这款。当然了还有款备用的也还不错。Sequel-Ace

postman api 工具

3. 办公套装

  • WPS 办公软件，microsoft 系列的替代品
  • Xmind Zen 强大的脑图工具
  • 印象笔记 笔记工具
  • 滴答清单 appStore 上直接下载
  • Typora 单机版笔记工具， md 的操作
  • 腾讯会议 视频会议的时候用到
  • Keka压缩包 开源的压缩包，满足日常解压缩
  • deepL 翻译软件 翻译工具
  • magnet : App Store 购买下载，用于桌面窗口的管理。可以一个桌面显示不同 App 的显示，使用起来还是蛮方便的。

  • TecentLemon 清理工具

  • 万年历 appStore 上直接下载

  • Better Menubar: AppStore 上直接下载

  • Microsoft Remote Desktop Beta win 的远程桌面
